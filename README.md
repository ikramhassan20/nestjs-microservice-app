## Description
A Nestjs Microservice App implements the communication to Microservice Client using event based MessagePattern. Handles the communication using Message_Service and event based approach. It will print the "Hello World printed" on the client microservice and print "Hello world" in the main Microservice App. 


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
